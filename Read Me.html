Setup Instructions

  • Required Tools:
    • WAMP
    • Netbeans
  • Procedure:
    • First, Install WAMP And Start WAMP Manager
    • Navigate To PHPMyAdmin
    • Create A Database Called 'Catalytics'
  • Configuration:
    • Open The Project In Netbeans And Navigate To 'MVCConfig.java'
    • Locate This Line 'prop.put("hibernate.hbm2ddl.auto", "validate");'
    • Change It To: 'prop.put("hibernate.hbm2ddl.auto", "create");'
    • This Tells Hibernate To Create Your Database For You. After The First Run, You Will Want To Change This Back To 'validate'
    • Setup Is Almost Complete! Run The Application For The First Time
    • Once Application Has Spawned, Navigate Back To PHPMyAdmin And Click The Table 'Roles'
    • Now Click The 'Insert Tab And Insert A New Role Named: "CLIENT", Then Repeat This Process And Create A New Role Named "ADMIN"
    • Now Go Register A New Account
    • Once Account Is Created Go Back To PHPMyAdmin And Click The Table 'Users_Roles'
    • Now Click The 'Insert Tab And Link Your User Account (1) To Roles 1 And 2 (Effectively Giving You Both 'CLIENT' & 'ADMIN' Roles
    • Setup Is All Complete! Make Sure To Change Hibernate's 'Create' Back To 'Validate'
